# fakethefunk

Simple python project for detecting upscaled low quality audio files, inspired by the fakethefunk app.

## project status 

Body is written but not tested or anything, that is for later. For now, use probe_file.py.  
`python -m fakethefunk.probe_file "path/to/file"`
